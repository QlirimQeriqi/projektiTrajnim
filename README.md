# Cineplex Stream (React)

Professional-style, frontend-only movie booking training app.

## Core Features

- Login with inline validation and error states
- Cinematic home page with auto-rotating hero slides
- Advanced movies catalog with:
  - Search (title/description/director)
  - Genre filter
  - Sort options
  - Min rating and max price sliders
- Detailed modal popup per movie (director, language, formats, showtimes, rating)
- Save movie to `My Plan`
- `My Plan` controls per movie:
  - Watch order: First / Second / Third
  - City: Prizren / Prishtine / Lipjan / Mitrovice
  - Showtime
  - Seat type (Standard / VIP)
  - Ticket quantity (1-8)
- Live booking totals (selected movies, tickets, estimated price)
- LocalStorage persistence
